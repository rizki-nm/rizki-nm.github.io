SurveyAsia

Platform Riset Online berbasis Website

Latar Belakang Project

SurveyAsia merupakan project yang saya kerjakan pada program Magang Bersertifikat Batch 3 Kampus Merdeka di PT Citi Asia Internasional sebagai Fullstack Developer. SurveyAsia dibuat dengan tujuan untuk membantu customer dalam pembuatan survei dan pengisian survei dengan berbagai macam pertanyaan. SurveyAsia memiliki role pengguna antara lain Researcher, Respondent, Admin, dan Publisher.

Proses Pelaksanaan Project

Pengembangan website SurveyAsia dilakukan dengan metode scrum. Pengerjaan dilakukan dengan total 16 Sprint. Dalam pelaksanannya, saya berkolaborasi dengan berbagai macam divisi seperti System Analyst, UI/UX Designer, Project Manager, Scrum Master, Data Engineer, Mobile Developer, dan IT Tester. Project dibangun menggunakan Framework Laravel, VueJS, Bootstrap, dan MySQL. Dalam pembuatan fitur menggunakan RESTful API sebagai komunikasi antara frontend dan backend. Project juga menggunakan Git dengan Trunk Based Development sebagai strategi git branching.
